Crust Wallet

Crust Wallet is a plug-in wallet. Users can manage their CRUs via Crust Wallet. It includes following features:

  • Interact with Crust DApps such as Crust Apps
  • Manage accounts
  • Transfer of tokens

Please visit Crust Wallet to install the Crust Wallet extension.
